# Contributing to FraudGuard

We welcome contributions! Here's how you can help.

## Prerequisites
*   Java 17+
*   Maven
*   MySQL

## Setup
1.  Fork the repository.
2.  Clone your fork: `git clone https://github.com/YOUR_USERNAME/Digital-Banking-Fraud-Detection-Simulation-Engine.git`
3.  Configure `src/main/resources/application.properties` with your MySQL credentials.
4.  Run `mvn clean install` to build the project.

## Running Tests
Run `mvn test` to execute the unit tests.

## Submitting Changes
1.  Create a new branch: `git checkout -b feature/my-new-feature`
2.  Commit your changes.
3.  Push to the branch: `git push origin feature/my-new-feature`
4.  Submit a Pull Request.
